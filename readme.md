## Features

- Listen to the stream
- User login
    - Favorite tracks
    - Manage favorites
    - Search for songs
    - Request songs (if the user can do so)


## Planned features

- Song history
- Display number of songs in queue
- [Bluetooth control](https://snag.gy/tiYgwn.jpg)
